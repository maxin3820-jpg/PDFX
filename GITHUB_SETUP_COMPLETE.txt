═══════════════════════════════════════════════════════════════════════════════
  QUICK START
═══════════════════════════════════════════════════════════════════════════════

  OPTION A: Build Debug APK (Easiest)
  ────────────────────────────────────

    1. Push to GitHub:
       
       git add .
       git commit -m "Setup GitHub Actions"
       git push origin main

    2. Go to Actions tab on GitHub

    3. Wait 3-4 minutes

    4. Download APK from Artifacts section

    5. Install on Android device

    ✅ Done! No configuration needed.


  OPTION B: Build Signed Release APK
  ───────────────────────────────────

    1. Create keystore:
       
       keytool -genkey -v -keystore release-keystore.jks \
         -keyalg RSA -keysize 2048 -validity 10000 \
         -alias pdfx-release

    2. Convert to Base64:
       
       Windows:  [Convert]::ToBase64String([IO.File]::ReadAllBytes("release-keystore.jks")) > keystore.txt
       Linux:    base64 -i release-keystore.jks -o keystore.txt

    3. Add 4 GitHub Secrets:
       
       Repository → Settings → Secrets → Actions → New secret

       KEYSTORE_BASE64      (contents of keystore.txt)
       KEYSTORE_PASSWORD    (your keystore password)
       KEY_ALIAS            (pdfx-release or your alias)
       KEY_PASSWORD         (your key password)

    4. Trigger workflow:
       
       Actions → Build APK → Run workflow → Select "release"

    5. Download signed APK from Artifacts

    📖 Detailed guide: .github/SIGNING_SETUP.md

  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

  Issue: Build fails
  Fix:   Check Actions tab → Click workflow run → View logs

  Issue: No artifacts
  Fix:   Ensure build succeeded (green checkmark), scroll to Artifacts

  Issue: APK won't install
  Fix:   Enable "Install from unknown sources" in device settings

  Issue: Keystore error
  Fix:   Verify all 4 secrets are set correctly (no typos)

  Issue: Gradle sync failed
  Fix:   Check build.gradle.kts syntax, verify dependencies
